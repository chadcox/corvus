# Code of Conduct

This project adopts the [Contributor Covenant](https://www.contributor-covenant.org),
version 2.1:

https://www.contributor-covenant.org/version/2/1/code_of_conduct/

In short: be respectful and constructive. Harassment and abusive behavior are
not tolerated.

## Reporting

Report unacceptable behavior privately to the maintainers via GitHub's private
vulnerability reporting (Security tab) or by contacting the repository owner.
Reports will be handled confidentially.
